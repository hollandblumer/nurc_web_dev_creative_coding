# NURC Web Development & Creative Coding Workshop

This repository contains a Next.js project used for the NURC web development and creative coding workshop. The site is statically generated and hosted using GitHub Pages.

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- GitHub Actions
- GitHub Pages

## Local Development

To run the project locally:

```bash
npm install
npm run dev
```
